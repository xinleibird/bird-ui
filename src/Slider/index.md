# Slider

## ⚙ defaultValue

- `[number]`

```jsx
import React from 'react';
import { Slider } from '@xinleibird/bird-ui';

export default () => {
  return (
    <>
      <Slider defaultValue={20} />
    </>
  );
};
```

`defaultValue` 默认初始值

## ⚙ range

- `[number]`

```jsx
import React from 'react';
import { Slider } from '@xinleibird/bird-ui';

export default () => {
  return (
    <>
      <Slider range={20} />
    </>
  );
};
```

`range` 取值范围
